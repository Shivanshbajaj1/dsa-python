# DSA in Python 🐍

A structured, beginner-friendly path through Data Structures & Algorithms using Python.

## How to use this repo

Each folder is one topic, in suggested order. Inside each folder:
- `README.md` — concept explanation + complexity notes
- numbered `.py` files — working implementations, runnable directly (`python 01_xxx.py`)
- an `exercises.py` (where present) — practice problems with `# TODO` markers and a `run_tests()` you fill in

Work through folders **01 → 10** in order. Each one builds on ideas from the last.

## Requirements

Just Python 3.8+. No external libraries needed.

## How to practice

1. Read the topic README.
2. Run the example file, step through it, modify values, break it on purpose.
3. Open `exercises.py`, solve each `TODO` without looking at the example file.
4. Re-implement the main file from scratch a day later, from memory.
